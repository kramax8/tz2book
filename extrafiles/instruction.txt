# Инструкция по запуску проекта.
# Проект сделан на yii2 basic

1. Открыть терминал в корне проекта
   и для того чтобы установить зависимости
   надо набрать эти команды:
   1.1 (composer self-update)
   1.2 (composer install)

2. В корне проекта есть папка
   под названием "extrafiles"
   и в этом папке есть dumpfile базы данных
   его надо импортировать.
   После этого проект должен работать.
